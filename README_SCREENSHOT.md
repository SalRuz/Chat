# Решение проблемы со скриншотами

## Проблема
`prismarine-viewer` не может найти файлы текстур из-за кириллицы в пути:
```
C:\Users\SalRuz\Desktop\files\майн бот\бот в тг майн бот\...
```

Node.js на Windows имеет проблемы с кодировкой путей к файлам, содержащих кириллицу.

## Решение

### Вариант 1: Переместить проект (рекомендуется)
Переместите папку с ботом в путь без кириллицы, например:
```
C:\minecraft-bot\
```

Затем:
```bash
cd C:\minecraft-bot
npm install
node bot
```

### Вариант 2: Использовать короткое имя пути
1. Откройте PowerShell от имени администратора
2. Выполните: `fsutil 8dot3name set C:\Users\SalRuz\Desktop\files\майн бот\бот в тг майн бот 0`
3. Используйте короткое имя: `C:\Users\SALRUZ~1\Desktop\FILES~1\МАЙН~1\БОТВ~1\bot.js`

### Вариант 3: Изменить кодировку Windows
1. Панель управления → Язык и стандарты → Дополнительные параметры даты и времени
2. Вкладка "Дополнительно" → "Изменить язык системы"
3. Поставьте галочку "Beta: Использовать Unicode UTF-8"
4. Перезагрузите компьютер

**Вариант 1 — самый надёжный!**
